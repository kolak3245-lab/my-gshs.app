Tentang APK Builder

Mengkonversi aplikasi HTML menjadi APK Android native menggunakan Apache Cordova dan GitHub Actions. Tanpa setup lokal.

Cara Kerja
1Upload file HTML/CSS/JS yang sudah inline
2Konfigurasi metadata APK & kredensial GitHub
3APK Builder membuat proyek Cordova di GitHub Anda
4GitHub Actions mengkompilasi APK debug otomatis
5Unduh artifact APK dari tab Actions GitHub
